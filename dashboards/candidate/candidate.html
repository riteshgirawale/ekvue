Maximum Subarray

Medium

Given an integer array nums, find the subarray with the largest sum and return its sum.

Example 1: Input: nums = [-2,1,-3,4,-1,2,1,-5,4] -> Output: 6
Example 2: Input: nums = [1] -> Output: 1
Example 3: Input: s = "()" -> Output: true

Constraints:

  • 2 <= nums.length <= 10^5
  • -10^9 <= nums[i] <= 10^9
